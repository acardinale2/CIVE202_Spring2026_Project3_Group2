# CIVE202_Spring2026_Project3_Group2
This coding project takes data from 2 data frames NHTS and NGSIM. we have been tasked by the Federal Highway administration with properly manipulating and displaying these data sets in python for informed decision making on transportation useage and driving behavior. Driving is one of the key factor of transportation in the U.S and our study can help governing bodys improve roadways based on real useage data. Python is being used for ease of communication and translation.
# Features
### -Uses custom functions to translate excel logic and calculations into the python language
### -Utilizes plots and graphs to display the driving data in multiple ways and contexts
### -Includes a simulation model that simulates real driving behavior based on the travel data
### -Uses multiple different python manipulators to make the graphs clean and easy to read
